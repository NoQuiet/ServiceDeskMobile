## ВВЕДЕНИЕ

**Service Desk Mobile** - мобильное приложение для управления заявками в службу технической поддержки. Приложение позволяет пользователям создавать заявки, отслеживать их статус, прикреплять фотографии и общаться со специалистами поддержки через комментарии.

### Основные возможности:
- Создание и отслеживание заявок
- Прикрепление фотографий к заявкам и комментариям
- Система комментариев для общения
- Оценка качества работы поддержки
- Управление пользователями (для администраторов)
- Назначение исполнителей (для администраторов и специалистов)

## РЕГИСТРАЦИЯ И ВХОД В СИСТЕМУ

### Вход в систему

1. **Запустите приложение** Service Desk Mobile
2. На экране входа вы увидите:
   - Иконку приложения (шестеренка)
   - Поля для ввода Email и Пароля
   - Кнопку "Войти"
   - Кнопку "Регистрация"
   - Кнопку "Как пользоваться?"

3. **Введите данные:**
   - Email: ваш адрес электронной почты
   - Пароль: ваш пароль

4. **Нажмите "Войти"**

**Примечание:** Если ваша учетная запись заблокирована администратором, вы увидите сообщение: *"Ваша учетная запись заблокирована. Обратитесь к администратору."*

### Регистрация нового пользователя

1. На экране входа нажмите **"Регистрация"**
2. Заполните обязательные поля:
   - **Email** - адрес электронной почты (используется для входа)
   - **Пароль** - минимум 6 символов
   - **Подтверждение пароля** - повторите пароль
   - **Имя** - ваше имя
   - **Фамилия** - ваша фамилия

3. Заполните дополнительные поля (опционально):
   - **Должность** - ваша должность
   - **Мобильный телефон** - контактный номер
   - **Внутренний телефон** - рабочий номер
   - **Этаж** - номер этажа
   - **Номер кабинета** - номер вашего кабинета

4. Нажмите **"Зарегистрироваться"**
5. После успешной регистрации вы будете автоматически перенаправлены на главный экран

**Примечание:** По умолчанию новым пользователям присваивается роль "Пользователь". Для получения роли "Специалист поддержки" или "Администратор" обратитесь к администратору системы.

## РОЛИ ПОЛЬЗОВАТЕЛЕЙ

В системе существует три роли с различными правами доступа:

### 1. Пользователь (user)
**Возможности:**
- Создание новых заявок
- Просмотр своих заявок
- Добавление комментариев к своим заявкам
- Прикрепление фотографий к заявкам и комментариям
- Оценка качества работы после решения заявки
- Удаление своих комментариев

### 2. Специалист поддержки (support)
**Возможности:**
- Все возможности пользователя
- Просмотр всех заявок в системе
- Взятие заявок в работу (назначение себя исполнителем)
- Изменение статуса заявок
- Добавление комментариев к любым заявкам
- Просмотр контактной информации авторов заявок

### 3. Администратор (admin)
**Возможности:**
- Все возможности специалиста поддержки
- Управление пользователями (просмотр, блокировка, разблокировка, удаление)
- Назначение и переназначение исполнителей на заявки
- Изменение ролей пользователей
- Полный доступ ко всем функциям системы

## РАБОТА С ЗАЯВКАМИ

### Создание новой заявки

1. На главном экране нажмите кнопку **"+"** (плюс) в правом нижнем углу
2. Откроется экран создания заявки
3. Заполните поля:
   - **Заголовок** - краткое описание проблемы (обязательно)
   - **Описание** - подробное описание проблемы (обязательно)

4. **Прикрепите фотографии** (опционально):
   - Нажмите кнопку с иконкой камеры
   - Выберите до 5 фотографий из галереи
   - Выбранные фото отобразятся в виде превью
   - Для удаления фото нажмите крестик на превью

5. Нажмите **"Создать заявку"**
6. После создания заявка появится в списке с статусом "Новая"

### Просмотр списка заявок

**Главный экран** отображает список всех доступных заявок:

**Для пользователей:**
- Отображаются только их собственные заявки

**Для специалистов и администраторов:**
- Отображаются все заявки в системе

**Информация на карточке заявки:**
- Номер заявки (#ID)
- Заголовок
- Описание (первые строки)
- Статус (цветная метка)
- Приоритет (цветная метка)
- Дата создания
- Автор заявки
- Исполнитель (если назначен)

**Фильтрация заявок:**
- Нажмите на иконку фильтра в верхнем правом углу
- Выберите фильтр:
  - **Все заявки** - показать все
  - **Мои заявки** - только созданные вами
  - **Назначенные мне** - где вы исполнитель

### Просмотр детальной информации о заявке

1. Нажмите на карточку заявки в списке
2. Откроется экран с подробной информацией:

**Информация о заявке:**
- Заголовок и описание
- Текущий статус
- Оценка (если есть)
- Дата создания
- Информация об авторе:
  - Имя и фамилия
  - Email
  - Должность
  - Мобильный телефон
  - Внутренний телефон
  - Этаж и номер кабинета
- Информация об исполнителе (если назначен)
- **Прикрепленные фотографии** (если есть)
- Список комментариев

### Статусы заявок

Заявка может находиться в одном из следующих статусов:

1. **Новая** (синий цвет) - заявка только что создана
2. **В работе** (оранжевый цвет) - специалист работает над решением
3. **Решена** (зеленый цвет) - проблема решена, ожидается оценка
4. **Закрыта** (серый цвет) - заявка закрыта

### Приоритеты заявок

1. **Низкий** (зеленый цвет)
2. **Средний** (желтый цвет)
3. **Высокий** (оранжевый цвет)
4. **Критический** (красный цвет)

### Изменение статуса заявки (для специалистов и администраторов)

1. Откройте заявку
2. Нажмите кнопку **"Изменить статус"**
3. Выберите новый статус из списка:
   - Новая
   - В работе
   - Решена
   - Закрыта
4. Нажмите **"Применить"**

### Взятие заявки в работу (для специалистов)

1. Откройте заявку без исполнителя
2. Нажмите кнопку **"Взять в работу"**
3. Вы будете назначены исполнителем
4. Статус автоматически изменится на "В работе"

### Назначение исполнителя (для администраторов)

1. Откройте заявку
2. Нажмите кнопку **"Назначить"** или **"Переназначить"**
3. Выберите специалиста из списка
4. Нажмите **"Назначить"**

### Оценка работы (для пользователей)

После того как заявка переведена в статус "Решена" или "Закрыта":

1. Откройте заявку
2. Нажмите кнопку **"Оценить работу"**
3. Выберите оценку от 1 до 5 звезд:
   - ⭐ - 1 звезда (плохо)
   - ⭐⭐ - 2 звезды (неудовлетворительно)
   - ⭐⭐⭐ - 3 звезды (удовлетворительно)
   - ⭐⭐⭐⭐ - 4 звезды (хорошо)
   - ⭐⭐⭐⭐⭐ - 5 звезд (отлично)
4. Нажмите **"Оценить"**

**Примечание:** Оценку можно поставить только один раз.

## КОММЕНТАРИИ

Комментарии позволяют общаться между пользователями и специалистами поддержки в рамках заявки.

### Добавление комментария

1. Откройте заявку
2. Прокрутите вниз до раздела "Комментарии"
3. В поле ввода введите текст комментария
4. **Прикрепите фотографии** (опционально):
   - Нажмите иконку камеры рядом с полем ввода
   - Выберите до 5 фотографий
   - Фото отобразятся под полем ввода
   - Для удаления фото нажмите крестик на превью
5. Нажмите иконку **отправки** (стрелка)
6. Комментарий появится в списке

**Цветовая индикация:**
- **Синий фон** - комментарий от пользователя (автора заявки)
- **Фиолетовый фон** - комментарий от специалиста поддержки

### Просмотр комментариев

В каждом комментарии отображается:
- Имя и фамилия автора
- Дата и время создания
- Текст комментария
- **Прикрепленные фотографии** (если есть) - горизонтальная галерея

### Удаление комментария

Вы можете удалить только свои комментарии:

1. Найдите свой комментарий (с иконкой корзины)
2. Нажмите на иконку **корзины**
3. Подтвердите удаление в диалоговом окне
4. Нажмите **"Удалить"**

**Примечание:** Удаление нельзя отменить.

## РАБОТА С ФОТОГРАФИЯМИ

Приложение поддерживает прикрепление фотографий к заявкам и комментариям.

### Прикрепление фотографий к заявке

**При создании заявки:**
1. На экране создания заявки нажмите иконку **камеры**
2. Выберите фотографии из галереи (до 5 штук)
3. Фото отобразятся в виде превью под кнопкой
4. Для удаления фото нажмите **крестик** на превью
5. Создайте заявку - фото загрузятся автоматически

### Прикрепление фотографий к комментарию

**При добавлении комментария:**
1. В поле ввода комментария нажмите иконку **камеры**
2. Выберите фотографии из галереи (до 5 штук)
3. Фото отобразятся под полем ввода
4. Для удаления фото нажмите **крестик** на превью
5. Отправьте комментарий - фото загрузятся автоматически

### Просмотр фотографий

**В заявке:**
- Прикрепленные фотографии отображаются в разделе "Фото (количество):"
- Фото показаны в виде горизонтальной галереи
- Размер превью: 120x120 пикселей
- Прокрутите галерею влево-вправо для просмотра всех фото

**В комментарии:**
- Фотографии отображаются под текстом комментария
- Горизонтальная галерея с прокруткой
- Размер превью: 100x100 пикселей

### Ограничения

- **Максимум 5 фотографий** на одну заявку или комментарий
- **Максимальный размер файла:** 5 МБ на одно изображение
- **Поддерживаемые форматы:** JPG, JPEG, PNG

## АДМИНИСТРИРОВАНИЕ

### Доступ к панели администратора

1. Войдите в систему как администратор
2. На главном экране нажмите иконку **меню** (три точки) в правом верхнем углу
3. Выберите **"Управление пользователями"**

### Просмотр списка пользователей

На экране управления пользователями отображается:
- Имя и фамилия
- Email
- Роль (Пользователь/Специалист/Администратор)
- Статус (Активен/Заблокирован)

**Цветовая индикация:**
- **Зеленый фон** - активный пользователь
- **Красный фон** - заблокированный пользователь

### Блокировка пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"Заблокировать"**
3. Подтвердите действие
4. Заблокированный пользователь не сможет войти в систему

**Примечание:** При попытке входа заблокированный пользователь увидит сообщение: *"Ваша учетная запись заблокирована. Обратитесь к администратору."*

### Разблокировка пользователя

1. Найдите заблокированного пользователя (красная карточка)
2. Нажмите кнопку **"Разблокировать"**
3. Пользователь снова сможет войти в систему

### Изменение роли пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"Изменить роль"**
3. Выберите новую роль:
   - **user** - Пользователь
   - **support** - Специалист поддержки
   - **admin** - Администратор
4. Нажмите **"Сохранить"**

**Примечание:** Изменения вступят в силу после повторного входа пользователя.

### Удаление пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"Удалить"**
3. Подтвердите удаление в диалоговом окне
4. Нажмите **"Удалить"**

**ВНИМАНИЕ:** Удаление пользователя необратимо! Все его заявки и комментарии останутся в системе, но будут отображаться без автора.

## РЕШЕНИЕ ПРОБЛЕМ

### Не могу войти в систему

**Проблема:** "Неверный email или пароль"
- Проверьте правильность ввода email и пароля
- Убедитесь, что Caps Lock выключен
- Попробуйте скопировать пароль из надежного источника

**Проблема:** "Ваша учетная запись заблокирована"
- Обратитесь к администратору системы
- Администратор должен разблокировать вашу учетную запись

### Не загружаются фотографии

- Проверьте подключение к интернету
- Убедитесь, что размер фото не превышает 5 МБ
- Проверьте формат файла (должен быть JPG, JPEG или PNG)
- Попробуйте выбрать другие фотографии

### Не отображаются заявки

- Потяните список вниз для обновления
- Проверьте активные фильтры
- Проверьте подключение к интернету
- Перезапустите приложение

### Не могу добавить комментарий

- Убедитесь, что поле комментария не пустое
- Проверьте подключение к интернету
- Попробуйте перезагрузить страницу заявки
